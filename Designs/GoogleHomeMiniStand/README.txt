# Print Settings

Printer Brand: Prusa
Printer: i3 MK2S
Rafts: Doesn't Matter
Supports: Yes
Resolution: 0,2
Infill: 20

Notes: 
You can choose high infill rate for "ghm_top.stl" for more strength.

If your printer can handle 60 degree overhang then you can print "ghm_top.stl" without support.
You should use support for "ghm_bottom_1.stl" for preparing to glue process as a next step.
