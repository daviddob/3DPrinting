# Summary

A customizable dowel holder for use in making filament dry boxes, filament spool racks, stands with racks for the Mosaic Palette+, etc.

I've included 2 premade stls.  The "-dry-box" I used to make a filament dry box to hold nylon / pva and it holds a 1/2" aluminum rod purchased at a hardware store.  The "-screw-mount" I used to hold the same 1/2" aluminum rod to make a stand + filament racks for my Palette+.

There are two parts to the holder.  A "flange" is (optionally) placed below the bracket holding the dowel.  The flange provides either gluing surface if you want to hot glue the bracket to a plastic container or provides optional holes to screw it to a wood frame or wall if you want to mount it with screws.

The "-dry-box" is generated by the default parameters of the customizer.

The "-screw-mount" is generated by changing the "flange_d" to 50 (and no other changes to the parameters are needed).
